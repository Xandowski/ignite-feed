## Getting Started

If you want set up the project locally.

### Prerequisites

- node or yarn

### Installation

1. Clone the repo

```sh
git clone {REPOSITORY_LINK}
```

2. Install NPM packages

```sh
npm install
or
yarn install
```

3. Run the app

```sh
npm run dev
or
yarn dev
```
